2018年终总结&2019年计划

2019-01-01
总结

2018年总结

看剧list+一句话总结

  • 《摇滚莫扎特》：年初的上海场和年末的天津场，在出坑的边缘试探
  • 《泽西男孩》：发微博被Luke点赞了
  • 《罗密欧与朱丽叶》：奶妈唱的太催泪了印象尤其深刻
  • 《深夜小狗离奇事件》：舞台设计非常棒
  • 《长靴皇后》：不知道为什么记不清了xx
  • 《吉屋出租》：no day but today
  • 《芝加哥》：看的时候正赶上《天朝渣男图鉴》被删，现实总比剧更戏剧化
  • 看了视频没有进剧院看过的：《巴黎圣母院》、《摇滚红与黑》、《1789》、《太阳王》、《德古拉》、《三个火枪手》、《Hamilton》

英语

  • 听力：“芝士派”的广播剧和电影共4期，打卡209天，累计精听101628个单词
  • 口语：“英语流利说”的懂你套餐3个月，学习时间50个小时，学习平均分92分
  • 阅读：“轻课阅读”1期，阅读 Animal Farm 和 The Call of The Wild 2本书，阅读时长86400分钟，阅读词量62314

python

  • 程序媛计划的四个python课程入了门，包括应用selenium做简单的爬虫
  • 后续的学习都是一边用一边学，有想做的先想好思路，想实现什么功能了就去查查别人是怎么实现的，哪个地方不会写了就去百度谷歌一下
  • 成果是github上的三个repositories, 可以用python解决一些工作生活中遇到的简单问题了，虽然写的乱七八糟的但是好歹能用了 (要求不高能跑就行
  • 野路子自己瞎写效率太低了，必须要系统的学一学了

deep learning

  • 完成了Coursera上的Deep Learning 专项课程的前三门

    1.Neural Networks and Deep Learning

    2.Improving Deep Neural Networks: Hyperparameter tuning, Regularization and Optimization

    3.Structuring Machine Learning Projects

  • 因为Andrew Ng大佬买的这个课，一开始以为会听不懂，后来发现讲的非常详细，还是很好理解的。刚上手最大的难度在于对于数学上术语的英文不了解，听多了熟悉了就好了（大概率还会遇到读中文教材和英文术语对不上的困难）。

  • 这个专项还有两门课，因为后来搬家等等一系列事情耽误了没有学完，计划在过年之前这两个月内学完。

博客

  • 6月份建了这个hexo+github page博客，8月份才真正用起来
  • 基本沦为英语笔记本(x
  • 后面需要学习一下怎么维护，首页social links的图标显示问题一直没解决
  • 希望以后能写一些高质量的文章

2019年计划

  1. 剧继续追起来，已经在计划内的有法扎和法红黑，还没官宣的NDDP如果引进了一定去看，其他的随缘

  2. 英语继续学下去，写作和阅读需要抓一抓，但是合适的课程有待探索

  3. 系统学习一下python，教材选用Mafnus Hetland 的 Begining Python—From Novice to Professional
  4. 完成了Coursera上的Deep Learning 专项课程的最后两门课，尽量自己写一些东西
  5. 发现现在写东西都是碎片化的思想也是碎片化的，新一年计划多阅读，每月最少两本书
  6. 加强锻炼，身体健康太重要了
